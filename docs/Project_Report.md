# Project Report — Edge Proctor

Summary

Edge Proctor is a browser-based proof-of-concept for on-device proctoring using MediaPipe FaceMesh. The app runs entirely client-side, keeping video frames local to the user's machine.

Design

- Frontend: React + Vite
- Face detection: MediaPipe FaceMesh (via CDN package)
- Gaze estimation: heuristic using eye landmark averages
- Attention: simple threshold-based timer that fires after 5s away

Limitations

- Gaze estimation is heuristic and not robust to all head poses or lighting.
- No server-side logging or tamper-proof audit trail included.

Next steps

- Replace heuristic gaze with calibrated model or use iris detection.
- Add configurable policies and session recording (with consent).
- Add e2e tests and CI workflow.
